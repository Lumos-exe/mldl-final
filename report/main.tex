\begin{abstract}
卷积神经网络具有局部感受野和平移等变等归纳偏置，通常能在中小规模图像数据集上稳定训练；视觉 Transformer 依靠自注意力机制建模长程依赖，但在小数据和轻量参数预算下容易出现优化困难或过拟合。本文围绕 CIFAR-100 图像分类任务，实现参数量接近的小型 CNN、Tiny ViT 和 CNN-Transformer 混合网络，并通过严格消融实验研究 attention 机制、Transformer 层数和 patch size 对混合结构的影响。实验重点不追求大模型复现，而是比较不同轻量结构在准确率、参数量和训练曲线上的差异，为课程设计场景下的混合视觉网络提供可复现实现和分析。
\end{abstract}

\noindent\textbf{关键词：} CIFAR-100；卷积神经网络；视觉 Transformer；轻量模型；消融实验

\section{引言}
CNN 通过卷积核共享和局部连接高效提取边缘、纹理和局部形状等视觉特征，是小图像分类任务中的强基线。ViT 将图像切分为 token 并使用多头自注意力建模 token 间关系，具备更灵活的全局依赖建模能力，但其缺少 CNN 的局部归纳偏置，通常需要更多数据、正则化和训练技巧。MobileViT\cite{mehta2022mobilevit} 和 MoCoViT\cite{ma2022mocovit} 等工作表明，将卷积模块与 Transformer 模块结合，有机会在移动端或轻量设置中获得更好的精度--复杂度平衡。

本文选择 CIFAR-100\cite{krizhevsky2009learning} 作为主任务。该数据集图像分辨率仅为 $32\times32$，但包含 100 个类别，比 CIFAR-10 更能体现模型表示能力差异。本文实现三类参数量接近的模型：小型 CNN baseline、Tiny ViT 对照模型、CNN+Transformer 混合模型，并围绕混合模型进行严格消融，以回答两个问题：第一，在相近参数预算下，混合结构是否优于单独 CNN 或单独 Transformer；第二，在该混合结构中，attention 交互、Transformer 层数和 patch size 分别如何影响性能。

\section{方法}
\subsection{CNN baseline}
CNN baseline 由三个卷积阶段组成，每个阶段采用 Conv-BN-GELU 的堆叠结构，并通过池化逐步降低空间分辨率。最后使用全局平均池化、LayerNorm 和线性分类器输出 100 类概率。该模型用于衡量只使用局部卷积归纳偏置时的表现，并通过宽度设置将参数量控制在与混合模型相近的范围。

\subsection{Tiny ViT}
Tiny ViT 参考 ViT\cite{dosovitskiy2021image} 的基本思想，将 $32\times32$ 图像划分为固定大小 patch，经卷积式 patch embedding 映射为 token 序列，加入 class token 和位置编码后输入 Transformer Encoder。该模型作为纯 Transformer 对照，用于观察在 CIFAR-100 这种小分辨率数据集上，缺少卷积前端时模型的优化和泛化表现。

\subsection{CNN-Transformer 混合模型}
混合模型先用两段卷积 stem 提取局部特征，将图像下采样为较小特征图；随后在特征图上切分 patch，输入轻量 Transformer Encoder 进行全局关系建模；最后对 token 做平均池化并分类。该设计保留了 CNN 对局部纹理的高效编码，同时让 Transformer 在更短 token 序列上建模全局语义关系。CNN baseline 和 Tiny ViT 分别作为只使用 CNN 与只使用 Transformer 的结构级对照。

\section{实验设置}
数据集使用 CIFAR-100，训练集 50,000 张、测试集 10,000 张。本文从训练集中固定划分 5,000 张作为验证集，用于选择最佳 checkpoint；官方测试集只在训练结束后评估一次。训练阶段采用随机裁剪、水平翻转、RandAugment 和标准化；验证和测试阶段仅进行标准化。优化器使用 AdamW，学习率采用 cosine 衰减，损失函数为带 label smoothing 的交叉熵。主实验训练参数量接近的 CNN baseline、Tiny ViT 和 Hybrid；严格消融实验均以 Hybrid 主模型为基准，每次只改变 attention 机制、Transformer 层数或 patch size 中的一个因素。

评价指标包括最终测试集 top-1 accuracy、可训练参数量、验证集 accuracy 曲线和单 epoch 耗时。所有实验固定随机种子，日志保存为 CSV，最佳模型依据验证集准确率保存。

\section{严格消融实验}
消融实验以 Hybrid 主模型为基准，每次只改变一个因素，其他设置保持不变。no-attention 实验保留 CNN stem、patch embedding、位置编码、FFN、投影层和分类头，只将 self-attention 替换为无 token-token 交互的前馈替代模块，用于判断 attention 交互本身是否有效。depth 实验只改变 Transformer Encoder 层数，用于判断增加或减少全局建模深度是否有帮助。patch size 实验只改变特征图切分粒度，用于判断 token 粒度对分类性能的影响。需要注意的是，严格消融不要求参数量恒定；当参数量或 token 数量随被改变因素自然变化时，应在结果分析中同时报告这些变化。

\section{结论}
本文完成了 CIFAR-100 上轻量 CNN、Tiny ViT 和 CNN-Transformer 混合模型的实现与对比。该课程设计的核心不是复现复杂大模型，而是在统一训练设置和相近参数预算下分析局部卷积特征与全局自注意力建模的互补关系。若实验结果显示 Hybrid 取得更高准确率，则说明卷积 stem 与轻量 Transformer 的组合在该任务上有效；若 Hybrid 未能超过参数匹配的 CNN 或 Tiny ViT，则应承认该串联式融合没有取得预期优势，并从融合方式、token 粒度、数据规模和训练策略角度分析原因。后续可进一步尝试更强的数据增强、知识蒸馏或更接近 MobileViT 的局部--全局--局部融合模块。
