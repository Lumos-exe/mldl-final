\begin{abstract}
卷积神经网络具有局部感受野和平移等变等归纳偏置，通常能在中小规模图像数据集上稳定训练；视觉 Transformer 依靠自注意力机制建模长程依赖，但在小数据和中等参数预算下容易出现优化困难或过拟合。本文围绕 CIFAR-100 图像分类任务，实现 CIFAR 版 ResNet-18、Small ViT 和参数预算接近的 ResNet-Transformer 混合网络，并通过严格消融实验研究 attention 机制、Transformer 层数和 patch size 对混合结构的影响。实验重点不追求大型模型复现，而是在统一训练与评估协议下比较不同中等规模结构在准确率、参数量和训练曲线上的差异，为课程设计场景下的混合视觉网络提供可复现实现和分析。
\end{abstract}

\noindent\textbf{关键词：} CIFAR-100；ResNet；视觉 Transformer；混合网络；消融实验

\section{引言}
CNN 通过卷积核共享和局部连接高效提取边缘、纹理和局部形状等视觉特征，是小图像分类任务中的强基线。ViT 将图像切分为 token 并使用多头自注意力建模 token 间关系，具备更灵活的全局依赖建模能力，但其缺少 CNN 的局部归纳偏置，通常需要更多数据、正则化和训练技巧。已有混合视觉网络并不是简单地在 CNN 后面追加 Transformer：MobileViT\cite{mehta2022mobilevit} 将 Transformer 作为卷积特征中的全局表示模块，形成局部--全局--局部的融合流程；CvT\cite{wu2021cvt} 将卷积引入 token embedding 和 Transformer 投影；CoAtNet\cite{dai2021coatnet} 则强调前期卷积与后期 attention 的有序组合；MoCoViT\cite{ma2022mocovit} 和 CMT\cite{guo2021cmt} 也从移动端和高效建模角度说明卷积与自注意力需要在结构中深度融合，而不是只在分类头前串接。

本文选择 CIFAR-100\cite{krizhevsky2009learning} 作为主任务。该数据集图像分辨率仅为 $32\times32$，但包含 100 个类别，比 CIFAR-10 更能体现模型表示能力差异。本文实现三类模型：CIFAR 版 ResNet-18、Small ViT 对照模型、ResNet-Transformer 混合模型，并围绕混合模型进行严格消融，以回答两个问题：第一，在接近参数预算和统一训练协议下，混合结构是否优于单独 CNN 或单独 Transformer；第二，在该混合结构中，attention 交互、Transformer 层数和 patch size 分别如何影响性能。

\section{方法}
\subsection{CNN baseline}
CNN baseline 采用适配 CIFAR 分辨率的 ResNet-18。模型使用 $3\times3$ 卷积作为 stem，不使用 ImageNet 版 ResNet 的大步长 stem 和 max pooling，以保留 $32\times32$ 小图像的空间细节。该模型用于衡量较强卷积结构在 CIFAR-100 上的表现。

\subsection{Small ViT}
Small ViT 参考 ViT\cite{dosovitskiy2021image} 的基本思想，将 $32\times32$ 图像划分为固定大小 patch，经卷积式 patch embedding 映射为 token 序列，加入 class token 和位置编码后输入 Transformer Encoder。该模型作为纯 Transformer 对照，用于观察在 CIFAR-100 这种小分辨率数据集上，缺少卷积前端时模型的优化和泛化表现。

\subsection{ResNet-Transformer 混合模型}
混合模型采用较窄的 CIFAR 版 ResNet 主干，以便在接近 ResNet-18 参数预算下为 Transformer 融合块留出容量。模型在第三阶段输出的 $8\times8$ 中层特征上插入一个 MobileViT 风格的局部--全局--局部融合块，主配置使用 $2\times2$ patch 将特征图转换为 16 个 token。具体而言，该模块先用局部卷积进一步提炼中层特征，再将特征图切分为 token 并输入 Transformer Encoder，以建模空间位置之间的全局关系；随后将 token 还原为空间特征图，并通过卷积与原始 CNN 中层特征融合，最后继续送入 ResNet 第四阶段和分类器。该设计不再用 Transformer 替换 ResNet 的后半段，也不只是接在最终分类头前，而是让全局关系建模参与后续卷积特征提取。ResNet-18 和 Small ViT 分别作为只使用 CNN 与只使用 Transformer 的结构级对照。

\section{实验设置}
数据集使用 CIFAR-100，训练集 50,000 张、测试集 10,000 张。本文从训练集中固定划分 5,000 张作为验证集，用于选择最佳 checkpoint；官方测试集只在训练结束后评估一次。训练阶段采用随机裁剪、水平翻转、RandAugment 和标准化；验证和测试阶段仅进行标准化。优化器使用 AdamW，学习率采用 cosine 衰减，损失函数为带 label smoothing 的交叉熵。主实验训练 ResNet-18、Small ViT 和参数预算接近的 ResNet-Transformer Hybrid，并统一报告准确率、参数量与训练曲线；严格消融实验均以 Hybrid 主模型为基准，每次只改变 attention 机制、Transformer 层数或 patch size 中的一个因素。

评价指标包括最终测试集 top-1 accuracy、可训练参数量、验证集 accuracy 曲线和单 epoch 耗时。所有实验固定随机种子，日志保存为 CSV，最佳模型依据验证集准确率保存。

\section{严格消融实验}
消融实验以 Hybrid 主模型为基准，每次只改变一个因素，其他设置保持不变。no-attention 实验保留局部卷积、patch embedding、位置编码、FFN、token 还原、卷积融合和分类头，只将 self-attention 替换为无 token-token 交互的前馈替代模块，用于判断 attention 交互本身是否有效。depth 实验只改变 Transformer Encoder 层数，用于判断增加或减少全局建模深度是否有帮助。patch size 实验只改变特征图切分粒度，用于判断 token 粒度对分类性能的影响。需要注意的是，严格消融不要求参数量恒定；当参数量或 token 数量随被改变因素自然变化时，应在结果分析中同时报告这些变化。

\section{结论}
本文完成了 CIFAR-100 上 ResNet-18、Small ViT 和 ResNet-Transformer 混合模型的实现与对比。该课程设计的核心不是复现复杂大模型，而是在统一训练设置下分析局部卷积特征与全局自注意力建模的互补关系。若实验结果显示 Hybrid 取得更高准确率，则说明在 ResNet 中层加入局部--全局--局部融合块对该任务有效；若 Hybrid 未能超过 ResNet-18 或 Small ViT，则应承认该融合方式没有取得预期优势，并从融合位置、token 粒度、数据规模和训练策略角度分析原因。后续可进一步尝试更强的数据增强、知识蒸馏或更接近 MobileViT 的多阶段融合结构。
